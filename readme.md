
### 此项目支持ES2015/TypeScript 开发

#### clone

```
https://github.com/xsstomy/webpack-egret
```

####  安装

```
cnpm install or npm install 
```

#### 启动

```
npm run start 
```

#### 发布

```
npm run build 
```

#### 目录结构

```
|---- index.html
|
|---- build
|
|---- libs
|
|---- resource
```
